参数:

  • inputs – 每个嵌段的体积分数，该输入可以是input.json文件、嵌段体积分数的向量以及’auto’，’auto’代表通过线性方程求解来自动修复密度。

  • species – 每种分子链的体积分数，字典形式（键为json文件中的SpecyID， 值为VolumeFraction）。

  • constraint – 对于自动求解的模式，可以增加额外的限制变量。
