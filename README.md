# Guide for Installment

## Python


`pip install SimuBox`

## Requirements

```
python=3.9
pytorch=1.8.2+cu111
pandas >= 1.4
scipy >= 1.10.0
matplotlib >= 3.6.2
numpy >= 1.19.5
opencv-contrib-python >= 4.7.0.72
opencv-python >= 4.5.3.56
pip i >= 3.4.18.6
```
