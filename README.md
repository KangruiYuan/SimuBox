# `Guide for Installment` 安装引导

## 请使用以下指令进行安装或者从`GitHub`中下载

`pip install SimuBox`

## `Requirements` 环境要求

```
python=3.9
pandas >= 1.4
scipy >= 1.10.0
matplotlib >= 3.6.2
numpy >= 1.19.5
opencv-contrib-python >= 4.7.0.72
opencv-python >= 4.5.3.56
```

- *SciTools*
  - **reader**

  - **generator**

    - VoronoiCell
  - **correlation**
    - Corr
- *SciPlot*
  - **compare**
