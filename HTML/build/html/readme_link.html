框架 Framework¶

安装引导 Installation¶

  1. From Github

git clone https://github.com/KangruiYuan/SimuBox.git

  1. From PyPI

相应版本一般落后于Github仓库

pip install SimuBox

TIPS: 安装后配置环境推荐使用poetry，依次执行以下python指令

pip install poetry

poetry install

使用示例 Demos¶

示例请见`/Demos/Scripts.ipynb`` <https://github.com/KangruiYuan/SimuBox/blob/main/Demos/Scripts.ipynb>`_

对于部分功能，进行了网页端可视化，在成功安装之后，可以通过以下指令启动网页服务：

python -m SimuBox.run

对于更为全面的功能使用，仍需通过代码实现。
